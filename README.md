# sf-terminal

ssh, sftp, scp, rsync terminal


# 安装

## 环境依赖

```shell
$ apt install expect
```

## sf-terminal

```shell
$ cd /usr/local

$ git clone https://github.com/xiangqians/sf-terminal.git

$ ln -s /usr/local/sf-terminal/sf_terminal.sh /usr/local/bin/sf_terminal.sh
```


# help

```shell
sf-terminal$ h
```
